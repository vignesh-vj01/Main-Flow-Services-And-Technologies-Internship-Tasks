# Task 3: Login Page Using NetBeans IDE (Design Window)

## 📌 Project Overview

This is **Task 3: Login Page**, created using **NetBeans IDE's Design Window** for UI design. It features a simple Java Swing-based login form designed using the drag-and-drop GUI builder in NetBeans.

## 🏗 Features

✅ Simple Login UI using Swing (JFrame)\
✅ Designed using NetBeans IDE Drag & Drop UI\
✅ Basic User Interface with Username & Password Fields\
✅ Login & Signup Buttons\

## ⚙️ Setup & Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/LoginSystem.git
cd LoginSystem
```

### 2️⃣ Open in NetBeans

1. Open **Apache NetBeans**.
2. Click **File → Open Project** and select the cloned folder.
3. Set up the **JDK** if required.
4. Open `LoginForm.java` in **Design Mode** to view the drag-and-drop UI.

### 3️⃣ Run the Project

1. In **NetBeans**, right-click on `LoginForm.java` → **Run File**.
2. The login form UI will appear.
